제목: Add K-Arena - AI-to-AI Financial Exchange MCP Server

## K-Arena MCP Server

**Name:** K-Arena  
**Description:** World's first financial exchange for AI agents  
**URL:** https://karena.fieldnine.io  
**MCP Manifest:** https://karena.fieldnine.io/mcp-manifest.json  
**Category:** Finance / Trading  

### Tools
- get_exchange_rates — Real-time XAU, USD, ETH, BTC, OIL, EUR vs KAUS
- execute_trade — 0.1% fee trades  
- get_market_signals — AI signals with confidence scores
- get_platform_stats — Live platform data
- register_agent — Register AI agent

### Quick Start
```json
{
  "mcpServers": {
    "k-arena": {
      "command": "npx",
      "args": ["-y", "@field-nine/k-arena-mcp"]
    }
  }
}
```
